# B.mumps — Blues Band ランディングページ 設計図

## 1. 概要

| 項目 | 内容 |
|------|------|
| サイト名 | B.mumps Official |
| 目的 | バンドの認知拡大・ライブ集客・SNS誘導 |
| ターゲット | ブルース好きの音楽ファン（20〜50代） |
| 技術スタック | HTML + Tailwind CSS（CDN）／1ファイル完結 |
| ダミー画像 | Unsplash プレースホルダー使用 |
| 公開先 | GitHub Pages |

---

## 2. デザインコンセプト

### トーン & ムード
- **ダーク × ゴールド** — 夜のライブハウスを想起させるカラーパレット
- スモーキーで渋い雰囲気、余白を活かしたタイポグラフィ重視のレイアウト
- フォント: 見出しに **セリフ体（Playfair Display）**、本文に **サンセリフ（Inter）**

### カラーパレット

| 用途 | カラー | Tailwind クラス例 |
|------|--------|-------------------|
| 背景（メイン） | `#0F0F0F` | `bg-[#0F0F0F]` |
| 背景（セクション交互） | `#1A1A1A` | `bg-[#1A1A1A]` |
| アクセント（ゴールド） | `#D4A843` | `text-[#D4A843]` |
| テキスト（メイン） | `#F5F5F5` | `text-[#F5F5F5]` |
| テキスト（サブ） | `#9CA3AF` | `text-gray-400` |
| ボーダー／装飾 | `#D4A843` 20% | `border-[#D4A843]/20` |

## 5. レスポンシブ対応方針

| ブレークポイント | 対応 |
|-----------------|------|
| `< 640px` (sm) | 1カラム、ハンバーガーメニュー、フォントサイズ縮小 |
| `640px – 1024px` (md) | 2カラム、ナビ表示 |
| `> 1024px` (lg) | フルレイアウト、最大幅 `max-w-6xl` で中央寄せ |

## 8. GitHub Pages 公開手順

1. GitHub に `B.mumps-LP` リポジトリを作成
2. `index.html` をプッシュ
3. Settings → Pages → Source: `main` / `/ (root)` を選択
4. 数分で `https://<username>.github.io/B.mumps-LP/` で公開

---

## 9. 実装時の注意事項

- すべてのスタイルは Tailwind のユーティリティクラスで完結させる（外部CSSファイル不要）
- ハンバーガーメニューの開閉は `<script>` タグ内に最小限のJSで実装
- 画像は `loading="lazy"` を付与してパフォーマンスを確保
- `<html lang="ja">` を設定し、日本語ページとして正しくマークアップ
- OGP メタタグを `<head>` に含め、SNS共有時の見栄えを確保
